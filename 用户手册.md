# 博客系统用户手册

## 1. 系统简介

### 1.1 系统概述
本博客系统是一个基于现代Web技术开发的内容管理平台，允许用户注册账号、发布文章、评论互动，并提供完善的管理员功能进行系统管理。系统采用前后端分离架构，提供流畅的用户体验和强大的内容管理能力。

### 1.2 主要功能特点
- **用户管理**：注册、登录、密码找回、个人信息管理
- **文章管理**：发布、编辑、删除、浏览文章，支持富文本编辑器和封面图片上传
- **评论系统**：发表评论、删除评论，支持评论分页和排序
- **搜索功能**：全局搜索文章标题、内容或作者昵称
- **管理员中心**：用户管理、文章管理、评论管理、系统统计概览
- **响应式设计**：适配不同屏幕尺寸的设备
- **安全认证**：基于JWT的用户认证机制，确保系统安全性

## 2. 快速开始

### 2.1 访问系统
在浏览器中输入系统网址即可访问博客系统。

### 2.2 注册账号
1. 点击页面顶部导航栏中的"注册"按钮
2. 在注册页面填写邮箱、昵称、密码等必要信息
3. 点击"获取验证码"按钮，等待系统发送验证码到您的邮箱
4. 输入收到的验证码，验证邮箱有效性
5. 点击"注册"按钮完成账号创建
6. 注册成功后系统将自动跳转至登录页面

### 2.3 登录系统
1. 点击页面顶部的"登录"按钮
2. 输入注册时使用的邮箱和密码
3. 点击"登录"按钮进入系统

## 3. 用户功能

### 3.1 用户注册
#### 3.1.1 注册流程
1. 访问注册页面（/register）
2. 填写以下信息：
   - 邮箱：必须使用有效的邮箱格式
   - 昵称：设置您的用户昵称（不可重复）
   - 密码：设置登录密码（长度要求）
   - 确认密码：再次输入密码确保一致
3. 点击"获取验证码"按钮，系统将发送验证码到您的邮箱
4. 输入收到的验证码
5. 点击"注册"按钮完成注册

#### 3.1.2 邮箱验证
系统会自动验证您输入的邮箱格式是否正确，并提供实时反馈。请确保输入有效的邮箱地址以接收验证码。

### 3.2 用户登录
#### 3.2.1 登录步骤
1. 访问登录页面（/login）
2. 输入您注册时使用的邮箱和密码
3. 点击"登录"按钮
4. 登录成功后将自动跳转到首页

#### 3.2.2 密码找回
如果您忘记了密码，可以通过以下步骤找回：
1. 在登录页面点击"忘记密码"
2. 输入您的注册邮箱
3. 获取并输入验证码
4. 设置新密码并确认
5. 点击"重置密码"完成操作

### 3.3 个人资料管理
#### 3.3.1 查看个人资料
登录后，点击页面顶部导航栏的个人中心，或者点击页面顶部的用户头像选择个人中心

#### 3.3.2 修改个人信息
在个人资料页面，您可以：
- 修改昵称
- 更新头像
- 修改简介

## 4. 文章功能

### 4.1 浏览文章
#### 4.1.1 文章列表
- 在首页可以查看所有文章的列表
- 支持按最新优先或最早优先排序
- 文章卡片显示标题、作者、发布日期、封面图片和摘要

#### 4.1.2 文章详情
- 点击文章标题或卡片进入文章详情页
- 文章详情页显示完整的文章内容、作者信息、发布日期

### 4.2 发布文章
#### 4.2.1 发布新文章
1. 登录后，点击页面顶部导航栏中的"写文章"按钮
2. 在文章编辑页面填写以下信息：
   - 标题：输入文章标题（必填）
   - 封面图片：点击上传按钮选择封面图片，支持常见图片格式（可选）
   - 内容：在富文本编辑器中编写文章内容，可以使用编辑器提供的各种格式化工具
3. 填写完成后，点击页面底部的"发布"按钮
4. 发布成功后，系统将自动跳转到文章详情页

#### 4.2.2 编辑文章
1. 在个人中心的我的文章处提供"编辑"按钮
2. 点击"编辑"按钮进入编辑页面
3. 修改文章内容后点击"更新文章"按钮

### 4.3 删除文章
- 在个人中心的我的文章处提供"删除"按钮
- 点击"删除"按钮并确认，即可删除文章

## 5. 评论功能

### 5.1 查看评论
在文章详情页的底部可以查看该文章的所有评论。评论按时间顺序显示，包括评论者昵称、评论时间和评论内容。

### 5.2 发表评论
1. 登录系统
2. 在文章详情页底部的评论输入框中输入评论内容
3. 点击"发表评论"按钮提交

### 5.3 删除评论
- 如果您是评论的作者或文章作者，可以在评论旁看到"删除"按钮
- 点击"删除"按钮并确认，即可删除评论

## 6. 搜索功能

### 6.1 全局搜索
- 在页面顶部的搜索框中输入关键词
- 选择搜索类型（可以是标题、内容或作者）
- 点击搜索按钮或按回车键
- 搜索结果将在首页显示

### 6.2 搜索筛选
- 搜索结果支持排序（最新优先/最早优先）
- 支持分页浏览搜索结果

## 7. 管理员功能

### 7.1 管理员登录（账号：admin@163.com  密码：123456）
管理员账号登录系统，登录后可在页面顶部头像下拉菜单处选择访问管理员中心。

### 7.2 管理中心概览
访问管理员中心可以看到以下功能模块：
- 统计概览：显示系统的各项统计数据
- 文章管理：管理所有文章
- 评论管理：管理所有评论
- 用户管理：管理所有用户

### 7.3 文章管理
在文章管理页面，管理员可以：
- 查看所有用户发布的文章列表，支持分页显示
- 编辑任意文章的标题、内容和封面图片
- 删除违规或不需要的文章
- 按作者、标题等条件筛选文章
- 查看每篇文章的详细信息

### 7.4 评论管理
在评论管理页面，管理员可以：
- 查看所有文章的评论列表，支持分页显示
- 删除不当或违规的评论
- 按用户或内容筛选评论
- 查看每条评论对应的文章和用户信息

### 7.5 用户管理
在用户管理页面，管理员可以：
- 查看所有注册用户的列表，支持分页显示
- 禁用/启用用户账号，控制用户访问权限
- 删除违规用户账号
- 查看用户的注册时间、角色等基本信息

### 7.6 系统统计
在统计概览页面，管理员可以查看：
- 总文章数和今日新增文章数
- 总评论数和今日新增评论数
- 总用户数和活跃用户统计
- 系统运行的各项关键指标

## 8. 常见问题

### 8.1 账号相关问题
- **Q: 注册时收不到验证码怎么办？**
  A: 请检查邮箱地址是否正确，查看垃圾邮件箱，或尝试重新发送验证码。如果问题持续，请联系系统管理员。

- **Q: 如何修改密码？**
  A: 通过"忘记密码"功能重置。修改密码需要邮箱验证码以确保账号安全。

- **Q: 我的账号被禁用了怎么办？**
  A: 账号被禁用通常是因为违反了系统规定。请联系系统管理员了解具体原因并申请恢复。

### 8.2 文章相关问题
- **Q: 文章封面图片有什么要求？**
  A: 支持JPG、PNG、GIF等常见图片格式，文件大小建议不超过2MB。

- **Q: 如何编辑已发布的文章？**
  A: 在个人中心我的文章处可看到“编辑"按钮，点击后即可修改文章内容。

- **Q: 我的文章发布后在哪里可以找到？**
  A: 您发布的文章会显示在首页文章列表中，同时您也可以在个人资料页面查看自己发布的所有文章。

### 8.3 评论相关问题
- **Q: 为什么我看不到删除评论的按钮？**
  A: 您只能删除自己发表的评论，管理员可以删除所有评论。

- **Q: 评论功能需要登录吗？**
  A: 是的，为了维护评论环境和追溯责任，发表评论需要先登录系统。

### 8.4 搜索相关问题
- **Q: 搜索结果不准确怎么办？**
  A: 请尝试使用更精确的关键词，或尝试不同的搜索类型（标题、内容、作者）。搜索功能会匹配包含关键词的文章。

- **Q: 为什么搜索结果显示不完整？**
  A: 搜索结果支持分页显示，请点击分页按钮查看更多结果。
